## Security Features
- Biometric authentication support
- 15-minute session timeout
- Role-Based Access Control (RBAC)
- Secure key management
- Compliance with:
  - ISO/IEC 27001:2022
  - NIST SP 800-53 Rev. 5
  - PCI DSS v3.2.1
